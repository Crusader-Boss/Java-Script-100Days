
# Primitive Data Types And Objects

Primitive data types are a set of basic data types in javascript.
object is non primitive datatype in javascript

These are 7 primitive datatypes in javascript (NN SS BB U) :-

 1. Null
 2. Number
 3. String
 4. Symbol
 5. Boolean
 6. Bigint
 7. Undefined
